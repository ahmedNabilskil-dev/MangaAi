# 🎯 **ENTITY DETAIL VIEWS - USER GUIDE**

## Overview

Your new chat interface now includes **comprehensive detail views** for every element in your manga project. These detail panels provide rich information and quick actions for all your project components.

## 🔍 **How to Access Detail Views**

### **From Side Panels**

1. **Project Structure Panel**

   - Click the **👁️ eye icon** next to any character name
   - Click the **👁️ eye icon** next to any chapter title
   - View detailed information without leaving the chat

2. **Template Library Panel**

   - Hover over any template card to reveal action buttons
   - Click the **👁️ eye icon** to view template details
   - Click the **✏️ edit icon** to use the template

3. **Generated Assets Panel**
   - Browse all your generated content
   - View asset metadata and usage information
   - Quick regeneration and variation actions

## 📋 **Detail Panel Features**

### **Character Detail View**

Shows comprehensive character information:

**Basic Information**

- Age, Gender, Physical attributes
- Body type, height, proportions

**Facial Features**

- Face shape, skin tone, eye details
- Nose type, mouth type, jawline

**Hair Details**

- Color, style, length, texture
- Special hair features

**Distinctive Features**

- Special markings, scars, accessories
- Physical mannerisms and posture

**Quick Actions**

- Edit character properties
- Duplicate character
- Delete character
- Share character details

### **Chapter Detail View**

Displays chapter structure and content:

**Chapter Information**

- Title, chapter number, status
- Publication status and tone

**Scene Breakdown**

- List of all scenes in the chapter
- Panel count for each scene
- Scene descriptions and context

**Metadata**

- Creation and update timestamps
- Chapter statistics

### **Template Detail Views**

#### **Outfit Templates**

- Category and subcategory
- Complete component breakdown
- Color, material, and pattern details
- Style and season information
- Preview images

#### **Location Templates**

- Environment type and setting
- Atmosphere and mood details
- Time of day and weather
- Usage examples

#### **Pose Templates**

- Body position breakdown
- Emotion and difficulty level
- Age group and gender compatibility
- Reference images

#### **Effect Templates**

- Effect type and intensity
- Application scenarios
- Visual style guides

## 🎨 **Visual Indicators**

### **Status Icons**

- 🟢 **Green dot**: Has generated image
- 👁️ **Eye icon**: View details action
- ✏️ **Edit icon**: Use/edit action
- 📋 **Copy icon**: Duplicate action
- 🗑️ **Trash icon**: Delete action

### **Badges**

- **Blue badges**: Component counts
- **Green badges**: Published status
- **Gray badges**: Draft/inactive status
- **Secondary badges**: Categories and tags

## 🚀 **How to Use Detail Views Effectively**

### **1. Character Development**

```
💡 Use Case: Developing a new character
1. Create character through chat
2. Click 👁️ on character name in structure panel
3. Review all physical attributes
4. Use "Edit" to refine details
5. Use "Duplicate" to create character variations
```

### **2. Template Selection**

```
💡 Use Case: Finding the perfect outfit
1. Go to Templates tab in side panel
2. Browse outfit categories
3. Hover over templates to reveal actions
4. Click 👁️ to see full component breakdown
5. Click ✏️ to apply template to character
```

### **3. Chapter Planning**

```
💡 Use Case: Reviewing chapter structure
1. Expand chapters in structure panel
2. Click 👁️ on chapter title
3. Review scene breakdown
4. Check panel distribution
5. Plan next scenes based on current structure
```

### **4. Template Comparison**

```
💡 Use Case: Comparing multiple templates
1. Open detail view for first template
2. Note key characteristics
3. Close and open detail view for second template
4. Make informed decision
5. Use chat to ask AI for recommendations
```
